# SINOPEC-02 baseline 结果

## 总体指标

- Raw macro-F1: `0.5932`
- Structured macro-F1: `0.6322`
- Raw weighted-F1: `0.9803`
- Structured weighted-F1: `0.9823`

## 分类表现

### Raw point-wise

- `0`: F1 `0.9894`
- `1`: F1 `0.6531`
- `2`: F1 `0.5303`
- `3`: F1 `0.2000`

### Structured decoding

- `0`: F1 `0.9899`
- `1`: F1 `0.7813`
- `2`: F1 `0.5984`
- `3`: F1 `0.1591`

## 结论

- 每井结构化解码明显改善了 `1`、`2` 两类关键点的识别
- `3` 类样本最少且不稳定，仍是当前 baseline 的主要短板
- 整体准确率很高，但这主要来自 `0` 类占比过大，因此 macro-F1 更有参考价值

## 下一步建议

1. 把点级分类改为候选点排序问题
2. 对 `3` 类做针对性重采样或代价敏感学习
3. 尝试 `XGBoost/LightGBM` 与轻量序列模型
4. 融合更多钻井过程特征
