# SINOPEC-02 消融实验结果

_用于解释特征工程中哪些部分真正带来了性能提升。_

---

## 实验设置

以 `RandomForest + 结构化解码` 为统一框架，分别去掉不同特征组，比较结构化 macro-F1 变化。

## 结果

| 配置 | 特征数 | Raw macro-F1 | Structured macro-F1 |
| --- | ---: | ---: | ---: |
| no_position | 35 | 0.5773 | 0.6476 |
| no_design | 29 | 0.5933 | 0.6369 |
| all_features | 38 | 0.5932 | 0.6322 |
| no_rolling | 26 | 0.5200 | 0.5419 |
| no_dynamic | 25 | 0.4797 | 0.4828 |
| raw_minimal | 12 | 0.3414 | 0.4016 |

## 解释

最重要的结论有三点。第一，动态变化率相关特征是当前任务中最关键的部分。去掉这些特征后，结构化 macro-F1 从 `0.6322` 下降到 `0.4828`，说明关键点识别本质上依赖轨迹变化趋势，而不仅仅是某个点的静态井斜角或方位角。第二，滚动统计特征也有明显贡献，说明仅靠逐点差分仍不够，模型需要局部窗口内的轨迹形态信息。第三，位置特征在当前实现下反而略有副作用，去掉后结果提升到 `0.6476`，这意味着井内相对位置可能引入了井型偏置。

设计轨迹特征仍然是有价值的，但它们的边际增益小于动态特征。去掉设计相关特征后，结构化 macro-F1 下降到 `0.6369`，说明设计轨迹偏差信息确实帮助模型理解“实际轨迹是否偏离预期”，但当前阶段真正支撑性能的仍然是轨迹变化本身。

## 结论

这组消融实验支持论文中的核心论断：井眼轨迹关键点识别不是简单的静态点分类问题，而是依赖轨迹局部动态和阶段切换模式的结构化识别问题。
